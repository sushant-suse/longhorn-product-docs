= バックアップボリュームの手動同期
:revdate: 2025-06-27
:page-revdate: {revdate}
:current-version: {page-component-version}

バックアップを作成した後、Longhornは元のボリューム（バックアップが基づいているボリューム）に対応するバックアップボリュームを作成します。バックアップボリュームは、同じボリュームの複数のバックアップを含むバックアップストア内のオブジェクトです。

以前のLonghornバージョンは、固定間隔で全てのバックアップボリュームをポーリングし、更新します。Longhornは、バックアップターゲットと手動でバックアップボリュームを同期する方法を提供します。

IMPORTANT: 同期を試みる前に、xref:snapshots-backups/volume-snapshots-backups/configure-backup-target.adoc[バックアップターゲット]を設定し、バックアップボリュームが作成されたことを確認する必要があります。バックアップターゲットとバックアップボリュームが存在しない場合、Longhornはエラーを返します。

* すべてのバックアップボリュームの同期：
 .. Longhorn UIで、*バックアップ*に移動します。
 .. *すべてのバックアップボリュームを同期*をクリックします。
* 単一のバックアップボリュームの同期：
 .. Longhorn UIで、*バックアップ*に移動します。
 .. バックアップボリュームを選択します。
 .. *バックアップボリュームを同期*をクリックします。

同期が成功したかどうかを確認するには、*バックアップ*画面でバックアップボリュームの名前をクリックします。
