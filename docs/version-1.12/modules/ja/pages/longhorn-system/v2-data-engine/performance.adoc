== パフォーマンス測定ツール

* https://github.com/yasker/kbench[KBench]:クラスターストレージパフォーマンスをベンチマークするために使用されます
* https://github.com/rancher/local-path-provisioner[ローカルパスプロビジョナー]:データディスクのベースラインパフォーマンスを測定するために使用されます

== Equinix (m3.small.x86)

* マシン:Japan/m3.small.x86
* CPU:Intel(R) Xeon(R) E-2378G CPU @ 2.80GHz
* RAM:64 GiB
* Kubernetes: v1.23.6+rke2r2
* ノード:3 (各ノードはマスターであり、同時にワーカーでもあります)
* OS:Ubuntu 22.04 / 5.15.0-33-generic
* ストレージ:1 SSD (Micron_5300_MTFD)
* ノード間のネットワークスループット（60秒間iperfでテストされた）：15.0 Gbits/sec
