== リリースラベル

各マイナーリリースは、次のマイナーリリースが安定するまでサポートされます。

* *Dev*:ソフトウェアは現在も活発に開発中であり、十分にテストされていません。以前のリリースからのアップグレードや、後のリリースへのアップグレードはサポートされていません。「Dev」リリースは通常スプリントの終わりに発生し、他の完了の定義が満たされた場合にも発生することがあります。リリースの非公式な性質を考慮し、ソフトウェアはテスト目的のみに使用してください。

* *Latest*:ソフトウェアはすべての検証およびテストの段階を通過しました。残っている既知の問題はすべて許容されると見なされます。ソフトウェアはテスト目的のみに使用してください。

* *Stable*:ソフトウェアは信頼性があり、重大な問題がないと見なされます。このリリースは一般的な使用に推奨されます。

* *EOL*:ソフトウェアは有用な期間の終了に達し、さらなるコードレベルのメンテナンスは提供されません。ライセンス契約の条件内でソフトウェアを引き続き使用することができます。安定したリリースへのアップグレードが推奨されます。

== 機能ラベル

**実験的**および**テクニカルプレビュー版**とラベル付けされた機能は、今後の革新を垣間見ることができ、新しい技術を環境内でテストする機会を提供します。

* *実験的*:この機能は未完成ですが、その基本的な機能は動作しています。制御された環境でのテストにより、既存の安定した機能と共存できることが示されています。欠落しているコンポーネントや進化する機能のため、この機能は運用環境での使用には推奨されません。

* *テクニカルプレビュー版*:この機能はほぼ完成しており、その機能が大幅に変更されることはないと予想されています。制御された環境でのテストにより、既存の安定した機能と共存できることが示されています。運用環境で使用する前に、この機能を十分に試用してください。

これらの機能には以下の制限があります:

* まだ開発中であり、機能が不完全であったり、不安定であったり、運用環境での使用には適していない場合があります。
* サポートされていません。
* 特定のハードウェアアーキテクチャでのみ利用できる場合があります。詳細および機能は変更される可能性があります。その結果、後続のリリースへのアップグレードが不可能になり、再インストールが必要になる場合があります。
* いつでも製品から削除される可能性があります。これは、たとえば、機能が顧客や市場のニーズを満たさない場合や、企業基準に準拠しない場合に発生することがあります。

[NOTE]
====
*実験的*および*テクニカルプレビュー版*の機能は通常、デフォルトで無効になっています。ドキュメントには、そのような機能を有効にし、構成する方法に関する情報が提供されています。
====

== 注意事項

注意事項は、追加または重要な情報を提供するテキストブロックです。各テキストブロックには特別なラベルとアイコンが付けられています。

* *注意*:追加情報は役立ちますが、必須ではありません。

* *重要*:製品の動作の変更など、特別な注意が必要な情報です。

* *ヒント*:タスクを完了するための代替方法など、役立つアドバイスや提案です。

* *注意*:特定の行動を進める前に考慮すべき情報、例えば発生する可能性のあるエラーです。

* *警告*:不可逆的な損害やデータ損失など、有害な結果に関する重要な情報です。
