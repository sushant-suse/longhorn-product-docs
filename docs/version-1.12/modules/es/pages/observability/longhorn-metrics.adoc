## CSI

El componente sidecar de CSI tiene métricas integradas para que los usuarios obtengan información sobre las operaciones de CSI. Las métricas de operaciones de CSI cubren el conteo total, el conteo de errores y la latencia de llamadas. Longhorn habilita las métricas añadiendo la bandera `--http-endpoint` para cada componente sidecar de CSI. Puedes usar https://github.com/prometheus-operator/prometheus-operator/blob/main/Documentation/api.md#podmonitor[PodMonitor de Prometheus] para recopilar estas métricas. 

|===
| Nombre | Puerto

| longhorn-csi-attacher
| 8000

| longhorn-csi-provisioner
| 8000

| longhorn-csi-resizer
| 8000

| longhorn-csi-snapshotter
| 8000
|===

Las métricas proporcionadas por el componente sidecar de CSI se presentan en un formato de histograma. Por ejemplo, se pueden obtener métricas observando el tiempo que tarda en crear un volumen de Longhorn para el PVC.
