= Características del motor de datos V2
:revdate: 2025-06-27
:page-revdate: {revdate}
:aliases: ["/spdk/features/_index.md"]
:current-version: {page-component-version}

* Soporte para plataformas AMD64 y ARM64
* Ciclo de vida del volumen (creación, montaje, desmontaje y eliminación)
* Volumen degradado
* xref:longhorn-system/v2-data-engine/features/node-disk-support.adoc[Gestión de discos en bloque]
* Gestión de réplicas huérfanas
* Creación, eliminación y reversión de instantáneas
* Copia de seguridad y restauración del volumen
* xref:longhorn-system/v2-data-engine/features/selective-engine-activation.adoc[Activación selectiva del motor de datos V2]
* xref:volumes/trim-filesystem.adoc[Trim del sistema de archivos]

Además de las características mencionadas anteriormente, se introducirán funcionalidades adicionales, como el ajuste del número de réplicas, la reconstrucción de réplicas en línea, etc. en versiones futuras.
