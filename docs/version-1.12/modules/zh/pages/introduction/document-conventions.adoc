== 发布标签

每个小版本在下一个小版本稳定之前都将得到支持。

* *Dev*：该软件正在积极开发中，尚未经过全面测试。不支持从早期版本升级或升级到后续版本。"Dev" 版本通常在冲刺结束时发布，并可能在满足其他完成定义时发布。鉴于该版本的非官方性质，仅将该软件用于测试目的。

* *Latest*：该软件已通过所有验证和测试阶段。所有剩余已知问题被视为可接受。仅将该软件用于测试目的。

* *Stable*：该软件被认为是可靠的，并且没有严重问题。建议一般使用此版本。

* *EOL*：该软件已达到其使用寿命的终点，不再提供代码级维护。您可以在许可协议的条款内继续使用该软件。建议升级到稳定版本。

== 功能标签

标记为**实验性**和**技术预览**的功能提供了即将推出的创新的预览，并提供了在您的环境中测试新技术的机会。

* *实验性*:该功能尚不完整，但其基本功能可以正常使用。在受控环境中的测试表明，它可以与现有的稳定功能共存。由于缺少组件和/或功能正在演变，因此不建议在生产环境中使用该功能。

* *技术预览*:该功能几乎完成，其功能预计不会发生重大变化。在受控环境中的测试表明，它可以与现有的稳定功能共存。在生产环境中使用之前，请广泛探索该功能。

这些功能具有以下限制：

* 仍在开发中，可能在功能上不完整、不稳定，或在其他方面不适合用于生产。
* 不支持。
* 可能仅适用于特定的硬件架构。细节和功能可能会发生变化。因此，升级到后续版本可能是不可能的，并且可能需要全新安装。
* 可以随时从产品中去除。例如，如果我们发现该功能不满足客户或市场需求，或不符合企业标准，则可能会发生这种情况。

*实验性*和*技术预览*功能通常默认禁用。文档提供了有关如何启用和配置此类功能的信息。

== 提示信息

提示信息是提供额外或重要信息的文本块。每个文本块都标有特殊标签和图标。

* *注*：额外的信息是有用的，但不是关键的。

* *重要*：需要特别注意的信息，例如产品行为的变化。

* *提示*：有用的建议和提示，例如完成任务的替代方法。

* *注意*：在进行特定操作之前必须考虑的信息，例如可能发生的错误。

* *警告*：关于有害后果的关键信息，例如不可逆转的损害和数据丢失。
