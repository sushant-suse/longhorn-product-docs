____
早期版本允许您使用第三方工具恢复 SUSE® 存储系统。
____

* xref:snapshots-backups/system-backups/restore-to-cluster-using-rancher-snapshot.adoc[使用 Rancher 快照恢复到包含数据的集群]
* xref:snapshots-backups/system-backups/restore-to-new-cluster-using-velero.adoc[使用 Velero 恢复到新集群]

____
当前的 SUSE® 存储版本包括开箱即用的系统备份和恢复功能。

* SUSE® 存储自定义资源将被备份并打包成一个系统备份文件，然后保存到远程备份目标。
* 之后，您可以选择一个系统备份以恢复到新集群或恢复到现有集群。
