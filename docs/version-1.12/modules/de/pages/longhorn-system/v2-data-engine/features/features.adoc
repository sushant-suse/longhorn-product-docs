= V2-Daten-Engine-Funktionen
:revdate: 2025-06-27
:page-revdate: {revdate}
:aliases: ["/spdk/features/_index.md"]
:current-version: {page-component-version}

* Unterstützung für AMD64- und ARM64-Plattformen
* Lebenszyklus von Volumes (Erstellung, Anbindung, Abtrennung und Löschung)
* Degradiertes Volume
* xref:longhorn-system/v2-data-engine/features/node-disk-support.adoc[Block-Datenträgerverwaltung]
* Verwaiste Replikatverwaltung
* Snapshot-Erstellung, -Löschung und -Rücksetzung
* Volume-Sicherung und -Wiederherstellung
* xref:longhorn-system/v2-data-engine/features/selective-engine-activation.adoc[Selektive Aktivierung der V2-Daten-Engine]
* xref:volumes/trim-filesystem.adoc[Dateisystem-Trim]

Zusätzlich zu den oben genannten Funktionen werden in zukünftigen Versionen weitere Funktionalitäten wie die Anpassung der Replikatzahl, der Online-Replikat-Neuaufbau usw. eingeführt.
