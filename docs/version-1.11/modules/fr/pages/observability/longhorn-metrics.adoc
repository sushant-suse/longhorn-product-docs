## CSI

Le composant sidecar CSI dispose de métriques intégrées permettant aux utilisateurs d'obtenir des informations sur les opérations CSI. Les métriques des opérations CSI couvrent le nombre total, le nombre d'erreurs et la latence des appels. Longhorn active les métriques en ajoutant le drapeau `--http-endpoint` pour chaque composant sidecar CSI. Vous pouvez utiliser https://github.com/prometheus-operator/prometheus-operator/blob/main/Documentation/api.md#podmonitor[PodMonitor de Prometheus] pour collecter ces métriques. 

|===
| Nom | Port

| longhorn-csi-attacher
| 8000

| longhorn-csi-provisioner
| 8000

| longhorn-csi-resizer
| 8000

| longhorn-csi-snapshotter
| 8000
|===

Les métriques fournies par le composant sidecar CSI sont présentées sous forme d'histogramme. Par exemple, vous pouvez obtenir des métriques en observant le temps nécessaire pour créer un volume Longhorn pour le PVC.
