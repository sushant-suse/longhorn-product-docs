= Recursos do V2 Data Engine
:revdate: 2025-06-27
:page-revdate: {revdate}
:aliases: ["/spdk/features/_index.md"]
:current-version: {page-component-version}

* Suporte para plataformas AMD64 e ARM64
* Ciclo de vida do volume (criação, anexação, desanexação e exclusão)
* Volume degradado
* xref:longhorn-system/v2-data-engine/features/node-disk-support.adoc[Gerenciamento de disco em bloco]
* Gerenciamento de réplicas órfãs
* Criação, exclusão e reversão de instantâneo
* Backup e restauração de volume
* xref:longhorn-system/v2-data-engine/features/selective-engine-activation.adoc[Ativação seletiva do V2 Data Engine]
* xref:volumes/trim-filesystem.adoc[Trim de Sistema de Arquivos]

Além dos recursos mencionados acima, funcionalidades adicionais como ajuste do número de réplicas, reconstrução de réplicas online e assim por diante serão introduzidas em versões futuras.
