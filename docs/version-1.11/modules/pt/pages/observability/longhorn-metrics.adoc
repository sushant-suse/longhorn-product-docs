## CSI

O componente sidecar CSI possui métricas incorporadas para que os usuários obtenham insights sobre as operações do CSI. As métricas das operações do CSI cobrem contagem total, contagem de erros e latência de chamadas. O Longhorn habilita as métricas adicionando a flag `--http-endpoint` para cada componente sidecar do CSI. Você pode usar https://github.com/prometheus-operator/prometheus-operator/blob/main/Documentation/api.md#podmonitor[PodMonitor do Prometheus] para coletar essas métricas. 

|===
| Nome | Portar

| longhorn-csi-attacher
| 8000

| longhorn-csi-provisioner
| 8000

| longhorn-csi-resizer
| 8000

| longhorn-csi-snapshotter
| 8000
|===

As métricas fornecidas pelo componente sidecar do CSI são apresentadas em um formato de histograma. Por exemplo, é possível obter métricas observando o tempo que leva para criar um Volume Longhorn para o PVC.
