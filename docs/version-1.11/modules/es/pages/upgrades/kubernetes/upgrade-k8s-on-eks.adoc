= Actualiza la versión de Kubernetes en Amazon Elastic Kubernetes Service (Amazon EKS)
:revdate: 2026-01-12
:page-revdate: {revdate}
:current-version: {page-component-version}

En {longhorn-product-name}, establece `replica-replenishment-wait-interval` en `0`.

Consulta https://docs.aws.amazon.com/eks/latest/userguide/update-cluster.html[Actualizando un clúster] para obtener instrucciones.

[NOTE]
====
Si has creado xref:nodes/managed-kubernetes/eks-managed-node-pools.adoc#_create_additional_volume[discos adicionales] para Longhorn, necesitarás añadir manualmente la vía del disco montado a la lista de discos de los nodos actualizados.
====
