= SUSE® Storageシステムのバックアップ
:revdate: 2025-06-27
:page-revdate: {revdate}
:current-version: {page-component-version}

____
以前のバージョンでは、サードパーティのツールを使用してSUSE® Storageシステムを復元することができました。
____

* xref:snapshots-backups/system-backups/restore-to-cluster-using-rancher-snapshot.adoc[Rancherスナップショットを使用して、データが含まれるクラスタに復元する]
* xref:snapshots-backups/system-backups/restore-to-new-cluster-using-velero.adoc[Veleroを使用して新しいクラスタに復元する]

____
現在のSUSE® Storageのバージョンには、すぐに使用できるシステムバックアップと復元が含まれています。

* SUSE® Storageのカスタムリソースはバックアップされ、単一のシステムバックアップファイルにまとめられ、リモートバックアップターゲットに保存されます。
* 後で、新しいクラスタに復元するためのシステムバックアップを選択するか、既存のクラスタに復元することができます。
____

* xref:snapshots-backups/system-backups/create-system-backup.adoc[Longhornシステムのバックアップ]
* xref:snapshots-backups/system-backups/restore-system.adoc[Longhornシステムの復元]
