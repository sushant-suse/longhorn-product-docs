* AMD64およびARM64プラットフォームのサポート
* ボリュームライフサイクル（作成、アタッチ、デタッチ、削除）
* 劣化したボリューム

* 孤立したレプリカ管理
* スナップショットの作成、削除、復元
* ボリュームのバックアップと復元
* xref:longhorn-system/v2-data-engine/features/selective-engine-activation.adoc[選択的V2データエンジンの有効化]
* xref:volumes/trim-filesystem.adoc[ファイルシステムのトリム]

上記の機能に加えて、レプリカ数の調整、オンラインレプリカの再構築などの追加機能が今後のバージョンで導入される予定です。
